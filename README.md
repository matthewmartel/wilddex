# WildDex

WildDex is a mobile-first web app where users upload photos of real animals, confirm the species, unlock animals in a numbered Dex, and view their sightings on a personal map.

## Current Status

WildDex currently has a frontend-only shell with placeholder data. Core routes exist for home, Dex, animal details, scan/upload, confirmation, map, regions, profile, login, and signup.

## MVP Features

- User accounts
- Numbered animal Dex
- Locked and unlocked animal states
- Photo upload
- AI animal identification suggestion
- Sighting map
- Animal detail pages

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Mapbox
- Gemini Vision API
